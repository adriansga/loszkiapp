# TODO — Loszki App

## Do zrobienia po powrocie

1. Otwórz CMD w folderze `loszki-app`
2. `npm install` — zainstaluje brakujące paczki (better-sqlite3, @anthropic-ai/sdk)
3. `npm run dev` — odpali serwer
4. Otwórz http://localhost:3000

## Żeby agent działał
- Otwórz `.env.local`
- Wpisz klucz: `ANTHROPIC_API_KEY=sk-ant-...`
- Klucz z: https://console.anthropic.com/

## Co jest gotowe
- Dashboard, Obiady, Zakupy, Spiżarnia, Budżet, Agent AI
- Baza SQLite z 34 daniami i 5 rachunkami (seed automatyczny)
- CLAUDE.md z pełną dokumentacją dla AI
